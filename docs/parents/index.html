    Parents

    Parents can sign up for a Schoork Parent account with their email address. From this account they can submit concerns, request appointments, and link to their children’s accounts.

    Users with the Admin permission can view a listing of all parent accounts.

    Parent accounts have a green navigation bar along the top of the screen.
